# 🌤️ Weather Tracker (SmartThings Edge Driver)

기상청 초단기실황 + 초단기예보 API를 활용하여 현재 위치의 기상 상황을 정밀하게 감지하고 스마트싱스 대시보드에 시각화합니다.

## ✨ 주요 특징

### 📊 종합 기상상황 표시
SKY(하늘상태)와 PTY(강수형태) 코드를 결합하여 직관적인 이모지와 함께 제공합니다.

| 조건 | 표시 예시 |
|------|----------|
| 맑음 | ☀️ 맑음 (0.0mm) |
| 구름많음 | 🌥️ 구름많음 (0.0mm) |
| 흐림 | ☁️ 흐림 (0.0mm) |
| 비 | 🌧️ 비 (2.5mm) |
| 비/눈 | 🌨️ 비/눈 (1.0mm) |
| 눈 | ❄️ 눈 (0.5mm) |
| 소나기 | ⛈️ 소나기 (5.0mm) |
| 빗방울 | 🌦️ 빗방울 (0.0mm) |
| 눈날림 | 🌨️ 눈날림 (0.0mm) |

### 🌧️ 강수 감지 (자동화용)
- 커스텀 `precipitation` capability를 사용하여 **SmartThings 내장 수분감지 알림 없이** 자동화 조건으로 활용합니다.
- 강수량(RN1)이 **0 초과**일 때 `wet`, 아니면 `dry`로 판단합니다.

### 🌡️ 실외 온습도
- **기온(T1H):** 기상청 관측 기온 (자동화 조건용)
- **습도(REH):** 기상청 관측 습도 (자동화 조건용, 실내 습도와 분리)

## 🔧 기술 구조

### API 이중 호출
초단기실황 API에는 SKY(하늘상태) 항목이 없으므로, 두 가지 API를 함께 사용합니다.

| API | 용도 | 주요 항목 |
|-----|------|----------|
| `getUltraSrtNcst` (초단기실황) | 현재 관측값 | PTY, RN1, T1H, REH |
| `getUltraSrtFcst` (초단기예보) | 하늘상태 | SKY |

### Capabilities

| Capability | 용도 | 비고 |
|-----------|------|------|
| `reasonmusic47804.weatherCondition` | 대시보드 종합 날씨 표시 | 이모지 + 강수량 |
| `reasonmusic47804.precipitation` | 자동화 강수 감지 (wet/dry) | 내장 알림 없음 |
| `temperatureMeasurement` | 기온 자동화 조건 | |
| `reasonmusic47804.outdoorHumidity` | 습도 자동화 조건 | 실내 습도와 분리 |
| `healthCheck` | 연결 상태 | online/offline |
| `refresh` | 수동 새로고침 | |

### 표시 로직
- **PTY 1~7:** 강수 형태 우선 표시 (강수량 무관)
- **PTY 0:** SKY 코드 기준 표시 (맑음/구름많음/흐림)
- **자동화 트리거:** RN1 > 0 일 때 `wet` 발행

## ⚙️ 설정 방법

### 1. 기상청 API 키 발급
[공공데이터포털](https://www.data.go.kr/)에서 **"기상청_단기예보 ((구)_동네예보) 조회서비스"**를 신청하고 인증키를 발급받습니다.

### 2. Edge Bridge 설치
허브의 외부 네트워크 제약을 우회하기 위해 로컬 프록시(Edge Bridge)가 필요합니다.

### 3. 스마트싱스 앱 설정

| 설정 | 설명 | 기본값 |
|------|------|--------|
| Bridge IP | Edge Bridge LAN IP | 192.168.1.1 |
| Bridge Port | Edge Bridge 포트 | 8088 |
| NX (격자 X) | 기상청 격자 X 좌표 | 84 (구미 상모) |
| NY (격자 Y) | 기상청 격자 Y 좌표 | 95 (구미 상모) |
| 기상청 API Key | 공공데이터포털 인증키 | - |
| 폴링 간격 | 날씨 조회 주기 (분) | 10 |
| Debug Log | 디버그 로그 활성화 | OFF |

> 격자 좌표 확인: [기상청 격자 좌표 조회](https://www.data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15084084)

## 💡 팁
- **UI 한글화:** 대시보드에 영어로 라벨이 표시될 경우, 앱 재시작 또는 기기 재스캔을 수행하면 한글화된 커스텀 UI가 적용됩니다.
- **좌표 설정:** 본인 거주지의 정확한 격자 좌표를 입력해야 정확한 동네 날씨가 반영됩니다.
- **알림 없는 자동화:** 기존 `waterSensor` 대신 커스텀 `precipitation` capability를 사용하여 불필요한 수분감지 푸시 알림이 발생하지 않습니다.
