# 🌧️ Rain/Snow Tracker_DaeGumi (SmartThings Edge Driver)

기상청 초단기실황 API를 활용하여 현재 위치의 종합 날씨 상태(기상상황, 기온, 습도, 강수량)를 실시간으로 감지하고 상세 정보를 제공하는 스마트싱스 엣지 드라이버입니다.

## ✨ 주요 특징
- **종합 기상상황 표시:** 단순히 강수 여부만 보여주는 것이 아니라, 기상청 SKY/PTY 코드를 결합하여 `☀️ 맑음`, `🌥️ 구름많음`, `☁️ 흐림`, `🌧️ 비` 등의 상세 상태를 이모지와 함께 표시합니다.
- **실외 온습도 추적:** 현재 위치의 기온(°C)과 습도(%)를 실시간으로 가져옵니다.
- **스마트한 대시보드 관리:** 실외 온습도가 실내 온습도계 탭에 섞이지 않도록 상세 화면과 자동화 조건에서만 나타나도록 최적화되어 있습니다.
- **정교한 강수 판단:** 강수 형태(PTY)와 실제 강수량(RN1)을 동시에 체크하여 0.1mm 이상의 실질적인 강수가 있을 때만 감지(`waterSensor wet`)합니다.
- **자동화 루틴 지원:** 
  - 비/눈 감지 시 창문 닫기 (waterSensor 조건)
  - 실외 온도/습도 기준에 따른 냉난방기 제어 (N도 이상/이하 조건)
- **Edge Bridge 연동:** 허브의 외부 네트워크 제약을 우회하기 위해 Edge Bridge(안드로이드 앱 등)를 프록시로 사용하여 안정적인 통신을 보장합니다.

## ⚙️ 설정 방법
1. **공공데이터포털**에서 "기상청_단기예보 ((구)_동네예보) 조회서비스" API 키를 발급받습니다.
2. 스마트싱스 앱 기기 설정에서 다음 항목을 입력합니다.
   - **기상청 API Key:** 발급받은 서비스 키 (인코딩된 키 권장)
   - **NX/NY:** 조회할 지역의 격자 좌표 (예: 구미 상모 84/95)
   - **Bridge IP/Port:** Edge Bridge가 실행 중인 기기의 IP와 포트(8088)
   - **폴링 간격:** 날씨 정보를 업데이트할 주기(분 단위, 기본 10분 권장)

## 💡 팁
- 실외 습도와 기상상황의 한글 라벨이 영어로 보일 경우, 앱을 완전히 종료 후 재시작하거나 기기를 삭제 후 다시 추가(스캔)하면 최신 한글 UI가 적용됩니다.
- 본 드라이버는 오픈 소스로 공유 및 수정이 자유롭습니다.
