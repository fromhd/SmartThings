# 🌤️ Weather Tracker (SmartThings Edge Driver)

기상청 초단기실황 API를 활용하여 현재 위치의 기상 상황을 정밀하게 감지하고 스마트싱스 대시보드에 시각화합니다.

## ✨ 주요 특징
- **종합 기상상황 표시:** SKY(하늘상태)와 PTY(강수형태) 코드를 결합하여 `☀️ 맑음`, `🌥️ 구름많음`, `🌧️ 비` 등의 상태를 직관적인 이모지와 함께 제공합니다.
- **정교한 강수 감지:** 단순히 비가 오는 것뿐만 아니라, 실제 강수량(RN1)이 **0.1mm 초과**일 때만 `waterSensor`를 `wet`으로 변경하여 오동작을 방지합니다.
- **자동화 최적화:** 
  - **비/눈 감지:** `waterSensor`를 통한 창문 닫기 루틴 구성 가능
  - **실외 온습도:** 기상청 데이터를 바탕으로 냉난방기 제어 가능 (실내 온습도와 섞이지 않게 처리)
- **Edge Bridge 연동:** 허브의 외부 네트워크 제약을 우회하기 위해 로컬 프록시를 사용하여 안정적으로 데이터를 수집합니다.

## ⚙️ 설정 방법

### 1. 기상청 API 키 발급
- [공공데이터포털](https://www.data.go.kr/)에서 **"기상청_단기예보 ((구)_동네예보) 조회서비스"**를 신청하고 일반 인증키(Encoding/Decoding)를 발급받습니다.

### 2. 스마트싱스 앱 설정
- **기상청 API Key:** 발급받은 서비스 키 입력
- **NX / NY:** 조회할 지역의 기상청 격자 좌표 ([좌표 확인 링크](https://www.data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15084084))
- **Bridge IP/Port:** Edge Bridge 서버 정보 (8088)
- **Polling Interval:** 업데이트 주기 (기본 5분 권장)

## 💡 팁
- **UI 언어:** 대시보드에 영어로 라벨이 표시될 경우, 앱 재시작 또는 기기 재스캔을 수행하면 한글화된 커스텀 UI가 적용됩니다.
- **좌표 설정:** 구미 상모동 기준 `84/95`와 같이 본인 거주지의 정확한 격자 좌표를 입력해야 정확한 동네 날씨가 반영됩니다.
