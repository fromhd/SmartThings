# 📊 Naver Stock Tracker (SmartThings Edge Driver)

네이버 금융의 실시간 시세 데이터를 활용하여 주가 정보를 제공하는 스마트싱스 엣지 드라이버입니다.

## ✨ 주요 특징
- **조회 주기 설정:** 기존 60초 고정 방식에서 탈피하여, 최소 **10초**부터 자유롭게 조회 주기를 설정할 수 있습니다.
- **간편한 연동:** 별도의 API 키 발급 없이 네이버 종목 코드만으로 빠르게 시세를 확인할 수 있습니다.
- **Edge Bridge 활용:** Edge Bridge를 통해 네이버 금융 서버에 안전하게 접근합니다.
- **수익률 계산:** 평단가를 입력하면 현재 시세 기준 수익률을 실시간으로 계산하여 표시합니다.

## ⚙️ 설정 방법
1. 스마트싱스 앱 설정에서 **종목 코드**(예: 064350)를 입력합니다.
2. **조회 주기**를 원하는 초 단위(예: 10)로 설정합니다.
3. Edge Bridge 서버의 IP 정보를 입력합니다. (비어있을 경우 mDNS로 자동 검색을 시도합니다.)
