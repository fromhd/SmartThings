# 📊 Naver Stock Tracker (SmartThings Edge Driver)

네이버 금융의 데이터를 활용하여 국내 주식 시세를 별도의 API 키 없이 간편하게 추적할 수 있는 스마트싱스 엣지 드라이버입니다.

## ✨ 주요 특징
- **간편한 설치:** 복잡한 API 발급 절차 없이 종목 코드만으로 즉시 사용 가능합니다.
- **고속 폴링 지원:** 최소 **10초** 단위의 유연한 조회 주기 설정 기능을 제공합니다.
- **야간 거래(NXT) 지원:** 정규장(KRX)뿐만 아니라 시간외/야간 거래 데이터도 스마트하게 감지하여 표시합니다.
- **수익률 계산:** 평단가를 입력하면 현재가 기준 실시간 수익률을 자동 계산합니다.
- **Edge Bridge 연동:** 로컬 네트워크의 프록시를 통해 네이버 금융 서버와 안전하게 통신합니다.

## ⚙️ 설정 방법

### 1. 스마트싱스 기기 설정
- **종목 코드:** 네이버 금융의 6자리 종목 코드 (예: `005930`)
- **조회 주기:** 원하는 초 단위 (기본 10초 권장)
- **평균 단가:** 본인의 보유 평단가
- **Bridge IP/Port:** Edge Bridge가 실행 중인 기기의 IP와 포트 (8088)

### 2. Edge Bridge 준비
- 본 드라이버는 외부 웹 API 호출을 위해 로컬 네트워크 내의 Edge Bridge(안드로이드 앱 또는 PC용 브리지)가 필요합니다.

## 📊 제공 데이터
- **Price:** `종목명: 현재가원` 형식의 대시보드 텍스트
- **Market:** 현재 거래 시장 (KRX / NXT)
- **Yield:** 실시간 수익률 (%)
- **Raw Price:** 숫자로 된 순수 현재가 (자동화 조건용)

## 💡 참고 사항
- 네이버 금융의 웹 API를 활용하므로 별도의 비용이 발생하지 않습니다.
- 잦은 요청은 IP 차단의 원인이 될 수 있으므로 적절한 조회 주기(10초 이상)를 권장합니다.
